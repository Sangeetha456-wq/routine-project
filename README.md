# routine-project
Mini project – Daily Habit Tracker



📊 Daily Habit Tracker
📌 Project Overview
The Daily Habit Tracker is a web-based mini project developed using HTML, CSS, and PHP. It helps users monitor, analyze, and improve their daily lifestyle habits such as sleep, study time, screen usage, and wake-up time. The system generates a performance score out of 100 using a rule-based evaluation method.

🎯 Features
User-friendly input form for daily habits
Rule-based scoring system (out of 100)
Performance classification (Excellent / Good / Needs Improvement)
Daily history tracking using file storage (data.txt)
Graphical analysis of user progress using Chart.js
Simple navigation between pages


🛠️ Technologies Used
HTML
CSS
PHP
JavaScript (Chart.js)
XML (for rules storage)

├── history.php        # Past records display
├── graph.php          # Graphical analysis
├── rules.xml          # Scoring rules
├── data.txt           # Stored user data

⚙️ How It Works
User enters daily habit details
PHP processes input using rule-based logic
System generates a score out of 100
Result is displayed with performance level
Data is stored in data.txt for history tracking
Graph page visualizes performance trends


📈 Output Example
Sleep: 7 hrs
Study: 5 hrs
Screen Time: 3 hrs
Score: 85/100 → Good 👍
